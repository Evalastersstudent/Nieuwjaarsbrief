Liefste Oma


  1. ... keer één is één,

    2. een jaartje is weer heen.


  2. ... keer twee is vier,

    3. een jaar vol plezier.


  3. ... keer drie is negen,

    4. voor elke dag een zegen.


  4. ... keer vier is zestien,

    5. 2016 was een mooi jaar, nu is het tijd om vooruit te zien


  5. ... keer vijf is twintig,

    6. hopelijk worden we van de drankjes niet te loslippig.


  6. ... keer zes is zesendertig,

    7. ik weet het, mijn gedichtje is een beetje geestig.


  7. ... keer zeven is negenenveertig,

    8. maar ik hou van jou voor eeuwig.


  8. ... keer één is acht,

    9. een zon die altijd lacht.


  9. ... keer negen is eenentachtig,

    10. lees men gedichtje straks nog eens rustig.


  10. ... keer tien is honderd,

    11. dan maken we 2017 samen heel bijzonder.

    Duizend keer BRAVO,

    Liefste Oma, je bent zo!

    Je kapoen Eva,

    Arendonk, 1 januari 2017
